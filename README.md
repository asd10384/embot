## Getting Started / 어떻게 시작하나요?

 * This discode bot is made of typescript.
 * 이 디스코드 봇은 타입스크립트로 제작되었습니다.

### Prerequisites / 선행조건

```

Node.js가 설치되어 있어야 합니다.
```

### Installing / 설치

```
npm install --save --force
```
